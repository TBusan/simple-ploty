Three-Contour Examples

three-contour is a contour plot library for Three.js, compatible with Plotly.js contour features. It uses the Marching Squares algorithm to generate contour lines and filled regions from 2D grid data.

Key Features:
  • Marching Squares algorithm with saddle point handling
  • Fill, Lines, Heatmap coloring modes
  • Catmull-Rom spline smoothing
  • Built-in color scales (Viridis, Jet, Hot)
  • Constraint contours support
