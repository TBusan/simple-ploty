自定义坐标

使用真实的物理坐标（如经纬度）作为坐标轴，而非默认的索引值。 

// 自定义坐标范围
const data = {
  z: zData,
  x: [100, 105, 110, 115, 120],  // 经度
  y: [20, 25, 30, 35, 40]           // 纬度
};

// 坐标转换到画布
const scaleX = width / (data.x[data.x.length-1] - data.x[0]);
const scaleY = height / (data.y[data.y.length-1] - data.y[0]);
