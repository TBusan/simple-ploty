热力图模式 (heatmap)

coloring: 'heatmap' 产生连续的热力图效果，颜色平滑过渡， 没有明显的等值线边界。适合展示连续变化的趋势。

当前配置： 

    

    

      
const zr = zrender.init(document.getElementById('main'));
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'heatmap',    // 热力图模式
    ncontours: 20           // 颜色分级数
  },
  colorscale: 'Jet'
});
zr.add(contour);
