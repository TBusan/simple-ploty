填充+线条模式 (fill + lines)

同时显示填充区域和等值线线条。先渲染填充多边形，再在其上绘制线条，形成清晰的等值线边界。 这是地形图、气象图等场景中常用的展示方式。 

// 填充+线条模式：先绘制填充，再绘制线条
// 1. 渲染填充区域
const polygon = new zrender.Polygon({
  shape: { points: fillPoints },
  style: { fill: color, stroke: 'none' }
});
zr.add(polygon);

// 2. 渲染线条（在填充之上）
const line = new zrender.Polyline({
  shape: { points: linePoints },
  style: { stroke: '#333', lineWidth: 1.5, fill: 'none' }
});
zr.add(line);
