无着色模式 (none)

coloring: 'none' 只显示等值线条，不进行任何颜色填充。 线条使用 line.color 指定的统一颜色。

等值线渲染区域 (none 模式)
当前配置： 

    

    

      
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'none',       // 无着色
    showlines: true
  },
  line: {
    color: '#333',        // 统一线条颜色
    width: 1.5,
    smoothing: 1.2
  }
});
