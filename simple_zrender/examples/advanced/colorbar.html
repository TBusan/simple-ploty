颜色条 (ColorBar)

通过 colorbar 配置颜色图例。可以设置标题、长度和厚度。 颜色条会自动显示在图表右侧。

等值线渲染区域
当前配置： 

    

    

      
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'fill'
  },
  colorscale: 'Viridis',
  colorbar: {
    title: {
      text: 'Temperature (°C)'
    },
    len: 0.8,         // 长度 (相对于绘图区域)
    thickness: 20      // 厚度 (像素)
  }
});
