Null 数据处理

数据矩阵中可以包含 null 值表示缺失数据。 等值线会自动绕过这些区域，不进行绘制。

数据预览： 

    

    

      
const zr = zrender.init(document.getElementById('main'));
const contour = new Contour({
  z: [
    [1, 2, null, 4, 5],
    [2, 3, 4, 5, 6],
    [null, 4, 5, 6, 7],
    [4, 5, 6, 7, 8],
    [5, 6, 7, 8, 9]
  ],
  contours: {
    coloring: 'fill'
  }
});
zr.add(contour);
