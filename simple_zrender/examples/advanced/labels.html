等值线标签

设置 contours.showlabels: true 可以在等值线上显示数值标签。 标签会智能放置在等值线的平坦区域，避免重叠。

当前配置： 

    

    

      
const zr = zrender.init(document.getElementById('main'));
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'fill',
    showlabels: true,     // 显示标签
    showlines: true,
    start: 0.2,
    end: 0.8,
    size: 0.15
  },
  line: {
    width: 1,
    smoothing: 1
  }
});
zr.add(contour);
