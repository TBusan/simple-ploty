交互事件

支持鼠标悬停、缩放、平移等交互。悬停时显示当前位置的数据值。 使用鼠标滚轮缩放，拖拽平移视图。

等值线渲染区域
交互说明： 
渲染完成后可进行交互操作
// 鼠标悬停事件
chart.on('mouseover', (event) => {
  const { x, y, value } = event;
  console.log(`位置: (${x}, ${y}), 值: ${value}`);
});

// 缩放事件
chart.on('zoom', (event) => {
  console.log('缩放级别:', event.scale);
});
