交互事件

支持鼠标悬停、缩放、平移等交互。悬停时显示当前位置的数据值。 使用鼠标滚轮缩放，拖拽平移视图。

交互说明： 
渲染完成后可进行交互操作
const zr = zrender.init(document.getElementById('main'));
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'fill'
  }
});
zr.add(contour);

// 鼠标悬停事件
zr.on('mouseover', (event) => {
  const { offsetX, offsetY } = event;
  console.log(`位置: (${offsetX}, ${offsetY})`);
});

// 缩放事件
zr.on('mousewheel', (event) => {
  console.log('滚轮事件', event);
});
