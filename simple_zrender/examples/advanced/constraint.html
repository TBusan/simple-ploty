约束等值线

通过 zmin 和 zmax 约束等值线的数值范围。 超出范围的值会被裁剪，颜色映射也会相应调整。

当前配置： 

    

    

      
const zr = zrender.init(document.getElementById('main'));
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'fill'
  },
  colorscale: 'Viridis',
  zmin: 0.2,    // 最小值约束
  zmax: 0.7     // 最大值约束
});
zr.add(contour);
