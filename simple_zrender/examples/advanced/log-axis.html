对数坐标轴

通过设置 xaxis.type: 'log' 或 yaxis.type: 'log' 启用对数刻度坐标轴。适用于数据跨度较大（如数量级差异）的场景。

当前配置： 

    

    

      
const zr = zrender.init(document.getElementById('main'));
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'fill'
  },
  xaxis: {
    type: 'log',              // 对数刻度
    range: [1, 100]          // 范围
  },
  yaxis: {
    type: 'log'
  }
});
zr.add(contour);
