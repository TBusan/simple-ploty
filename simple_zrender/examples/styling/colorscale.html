自定义色阶

通过 colorscale 自定义颜色映射。可以是预定义的色阶名称（如 'Viridis'）， 或自定义的颜色数组，每个元素为 [value, color]，value 范围 0-1。

等值线渲染区域
色阶预览： 

    

    

      
// 自定义色阶
const customColorScale = [
  [0, '#000080'],     // 深蓝
  [0.25, '#0066cc'],  // 蓝
  [0.5, '#00ff00'],   // 绿
  [0.75, '#ffcc00'],  // 黄
  [1, '#ff0000']      // 红
];

const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: { coloring: 'fill' },
  colorscale: customColorScale
});
