平滑参数

通过 line.smoothing 控制等值线的平滑程度。取值范围 0-1.3：
• 0 - 无平滑，显示原始折线
• 1 - 默认平滑（推荐）
• 1.3 - 最大平滑，曲线更圆滑

等值线渲染区域
当前配置： 

    

    

      
const contour = new Contour({
  z: data.z,
  x: data.x,
  y: data.y,
  contours: {
    coloring: 'lines'
  },
  line: {
    smoothing: 1.3    // 0-1.3，值越大越平滑
  }
});
